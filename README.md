# tarpit
ai scraper tarpit

install dependencies:

```bash
sudo apt update
```
```bash
pip install beautifulsoup4 --break-system-packages
```

clone in:
```bash
git clone https://github.com/ekomsSavior/tarpit.git
cd tarpit
chmod +x tarpit.py
```

run it: 
wizard mode/ ez setup walkthru:

```bash
python3 tarpit.py --wizard
```
